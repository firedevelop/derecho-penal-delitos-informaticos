# Derecho Penal Delitos Informáticos
María José López Etxebarria tiene una amplia experiencia en la defensa de personas acusadas de delitos informáticos, como el acceso ilícito a sistemas informáticos, el robo de datos, la falsificación de documentos electrónicos y el ciberacoso. También ha asesorado a empresas y organizaciones sobre cómo prevenir y detectar delitos informáticos.
